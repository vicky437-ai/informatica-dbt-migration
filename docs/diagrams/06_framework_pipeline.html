infa2dbt Framework Pipeline

End-to-end flow from Informatica XML to Snowflake-native dbt project

1
Discovery & Inventory
infa2dbt discover
Input
Informatica PowerCenter XML directory
Output
XML inventory + source schema metadata
Sources
Snowflake INFORMATION_SCHEMA, XML extraction, manual JSON
Key data
Mapping names, workflows, repositories, transformation types, source tables
discovery/xml_inventory.py discovery/schema_discovery.py
2
Analysis & Complexity Scoring
infa2dbt convert (phase 1)
Input
Parsed XML (Repository model)
Output
Complexity scores, model strategy, enriched mappings
Scoring
11 dimensions: sources, targets, transforms, lookups, expressions, etc.
Strategies
DIRECT (0-30), STAGED (31-55), LAYERED (56-80), COMPLEX (81-100)
analyzer/complexity.py analyzer/multi_workflow.py analyzer/transformation_registry.py
3
LLM-Powered Conversion
infa2dbt convert (phase 2)
Input
Enriched mappings + complexity analysis
Output
dbt SQL models, YAML schemas, macros, tests
LLM
Snowflake Cortex (claude-4-sonnet) with self-healing loop
Chunking
Context-preserving chunks for large mappings (128K token limit)
generator/llm_client.py generator/prompt_builder.py generator/response_parser.py generator/post_processor.py generator/quality_scorer.py chunker/context_preserving.py cache/conversion_cache.py
4
Project Assembly & Merge
infa2dbt convert --mode merge
Input
Generated files per mapping
Output
Unified dbt project with deduplicated sources
Key ops
Source consolidation, conflict resolution, project structure
Layers
staging/ (views) → intermediate/ (views) → marts/ (incremental/table)
merger/project_merger.py merger/source_consolidator.py merger/conflict_resolver.py generator/dbt_project_generator.py
5
Validation
infa2dbt validate
Input
Assembled dbt project
Output
Validation results (compile/run/test status)
Static
SQL syntax, YAML structure, ref/source integrity
Live
dbt compile → dbt run → dbt test (Snowflake warehouse)
validation/dbt_validator.py validator/sql_validator.py validator/yaml_validator.py validator/project_validator.py
6
Version Control
infa2dbt git-push
Input
Validated dbt project
Output
Git commit + push to remote
Operations
init, branch, commit, push, remote management
Integration
Snowflake Git Repository for pull-from-Git deployment
git/git_manager.py
7
Snowflake Deployment
infa2dbt deploy
Input
dbt project (local or Git)
Output
Snowflake-native dbt project object + optional TASK
Direct
snow dbt deploy → CREATE DBT PROJECT
Git
CREATE GIT REPOSITORY → CREATE DBT PROJECT FROM @repo
deployment/deployer.py
